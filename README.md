

#### mesa项目介绍
通用化的六处理任务平台


#### 开发语言
java(1.8) \scala (2.11.8) \python 3.5


#### 技术栈
* jstorm
* opentsdb
* spark
* hbase
* kafka
* zookeeper
* kudu



##### 路线图
* 整理\梳理基础类型任务
* 实现任务流的管理
* 可视化管理所有的任务流
* 集成任务调度\任务监控管理功能


#### jstorm-plugin
* ZookeeerSpout ``实现全局的动态通知功能``
* RabbitSpout ``扩展实现rabbit mq队列``
* KuduBolt ``增加kudu数据的写入``
